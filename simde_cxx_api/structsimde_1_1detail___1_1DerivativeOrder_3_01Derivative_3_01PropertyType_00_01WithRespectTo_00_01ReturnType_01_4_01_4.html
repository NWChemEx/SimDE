Detailed Description

template<typename PropertyType, typename WithRespectTo, typename ReturnType>
struct simde::detail_::DerivativeOrder< Derivative< PropertyType, WithRespectTo, ReturnType > >

Deduces the derivative order of a property type.

This partial specialization is used when the template parameter for DerivativeOrder is a specialization of Derivative. The resulting class will have a member value set to the order of the derivative, i.e., 1 if the property type is for a gradient, 2 if the property type is for a Hessian, etc.
