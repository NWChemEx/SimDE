template<typename T>
struct simde::detail_::TensorRepTraits< T >

Struct used to synchronized input types across the tensor representation component of SimDE.

Template Parameters
TThe orbital space we are generating types for. Assumed to be one of (or implicitly convertible to one of):
  • type::ao_space,
  • type::derived_space,
  • type::tot_derived_space, or
  • type::independent_space
