Detailed Description

template<typename T>
struct simde::detail_::IsDerivative< T >

Deduces if T is a specialization of Derivative.

This is the primary class template for the IsDerivative type trait. This template is used when T is not a specialization of Derivative. As such it always contains a static constexpr member value set to false. This class template should be partially specialized for the scenario when T is a specialization of Derivative.

Template Parameters
TThe type to inspect. This trait will determine if T is a specialization of Derivative. T is expected to be an unqualified type.
